# Ninja vs Zombies 
## Side Scroller made with Pygame

### To install and play follow the instructions below:

#### Install Pygame
1.  Verify Python installed with command: `python --version` 
2.  Verify pip installed with command: `pip --version`
3. Install globally: `pip install pygame`
- If error occurs from step 3 or you would like to install in home dir instead of globally, continue to step 4 else continue to check step 5
4. Install in home directory vs globally: `python3 -m pip install -U pygame --user` 
5.  To verify install run this command: `python3 -m pygame.examples.aliens`

#### Install Ninja vs Zombies
1. Click on the green Code button
2. Decide between cloning with the HTTPS or SSH (copy link) or download ZIP (then extract not clone)
3. For HTTPS or SSH, goto your terminal and navigate to where you wish the files to be.<br>
&nbsp;&nbsp;&nbsp;&nbsp;- HTTPS run this command: `git clone https://github.com/JamakaDev/pygame-side_scroller.git`<br>
&nbsp;&nbsp;&nbsp;&nbsp;- SSH run this command: `git clone git@github.com:JamakaDev/pygame-side_scroller.git`<br>
4. For downloaded ZIP files, extract the files into a main folder
5. Open the new folder in your IDE or Text-Editor to run
